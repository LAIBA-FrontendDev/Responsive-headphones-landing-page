Responsive Headphones Landing Page
A modern and responsive landing page for a fictional headphones brand. This project highlights product features, specifications, and stylish design using only HTML and CSS.
Live Demo
https://laiba-frontenddev.github.io/Responsive-headphones-landing-page/
Features
- Hero section with product image and call-to-action
- Specs section with Bluetooth, battery, and mic details
- Product case showcase with description
- Discount banner with “Shop Now” CTA
- Product gallery with multiple color options
- Footer with product links, support, and social media icons
- Fully responsive layout for mobile and desktop
Technologies Used
- HTML5
- CSS3
- Git & GitHub
- GitHub Pages (for deployment)
Getting Started (Optional)
To run this project locally:
git clone https://github.com/laiba-frontenddev/Responsive-headphones-landing-page.git
cd Responsive-headphones-landing-page
open index.html
